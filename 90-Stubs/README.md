# 90-Stubs

Auto-generated proper-noun stubs. Each file is a one-line **stub** for a person, faction, or place mentioned somewhere in the corpus that didn't already have its own page.

**Triage workflow:**

1. Open a stub.
2. Decide: real entity? If not, delete the file.
3. If real, move the file to the correct home (`20-Factions/<slug>/summary.md`, `20-Factions/<faction>/people/<slug>.md`, `30-Places/.../<slug>.md`, etc.) and update the `description` frontmatter and body.
4. Update the relevant folder README index.
