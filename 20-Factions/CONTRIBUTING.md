# Contributing — Factions

## Per-faction layout

```
<slug>/
  summary.md
  people/<npc-slug>.md
  associations/      # items, vehicles, holdings the faction guards
  activity/
    week_N/<action-slug>.md   # ongoing actions; resolved via d20 skill check vs difficulty
```

## Slug rules

- Drop leading `the-`.
- Strip parenthetical house names (record as `aka`).
- Worshipful Companies prefix `wc-`.
