# Salvage Operation

[Chief Artificer Glim-Nimblewhiz](../../people/glim-nimblewhiz.md), still rebuilding from [last week's catastrophic prototype explosion](../../../lantaner-concession/activity/week_3/build-the-prototype.md), authorizes a salvage run on the abandoned Steel Watch Foundry in [Gray Harbour](../../../../30-Places/sword-coast/baldurs-gate/lower-city/gray-harbour.md) for the alloys his replacement design needs. The foundry is not as defunct as the cogwork team assumed. A handful of damaged but still-operational Steel Watchers attack the salvage crew on sight, treating them as intruders to a valid security perimeter. The gnomes retreat in chaos, leaving precision tools and partial schematics behind, recovering nothing usable. **The Cogwork Courier project is set back significantly — possibly for the rest of the season.** The functional Steel Watchers' continued existence inside the foundry is the more important and more terrifying revelation; Glim files an urgent report up to the Concession's senior council and quietly to the Sodality.
