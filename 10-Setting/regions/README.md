# Regions

Geographic features at setting scale: forests, mountain ranges, river systems, and other features whose extent and significance exceeds what fits inside any one folder under `30-Places/`. The local-face view of each feature lives under `30-Places/`; this folder holds the wider treatment.

## Index

- [cloakwood.md](cloakwood.md) — vast forest south of Baldur's Gate; Sword Coast timber reserve; fey-real; in active war.
- [wood-of-sharp-teeth.md](wood-of-sharp-teeth.md) — east of the Trade Way; severed from the natural world by the Absolute Crisis; generates psychic horror.
