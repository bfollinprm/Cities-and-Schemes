# Contributing — Setting

## Layout

```
polities/
  <polity>.md          # governance system as world-fact (e.g., waterdeep-government)
history/
  <topic>.md           # narrative historical arc (synthesis, not dated event list)
```

## Notes

- Polity files describe the system of rule, not the geography of the place. Geography lives in `30-Places/`.
- History files here are synthetic narrative. Discrete dated events go in `40-Timeline/history.md`.
- Add a folder `README.md` only when the folder grows to two or more files.
