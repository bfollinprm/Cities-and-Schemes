# Faction Operations Contents

## What Lives Here

This folder tracks active faction maneuvers, operations, and week-by-week questing so the GM can treat the city like a living chessboard.

## How To Use It

Start with the week-specific subfolders to see which factions are moving, then use `factions/` to drill into the notes that matter.

## Expected Notes

- `factions/` for one note per faction across the workbook
- `week-0/` through `week-7/` for the opening run of faction operations
