# Contributing — Places

## Geographic layout

```
<region>/
  README.md
  polities.md
  demographics.md
  <city>/
    README.md
    factions.md
    character.md
    neighbors.md
    <district>/
      README.md
      neighbors.md
      character.md
      <location>/
        README.md
        factions.md
```

Higher-level geographic concepts are not just holding folders. A region, city, district, or site can have its own topical markdown files as needed, in addition to child folders.

Geography depth is flexible. Collapse levels for small places, expand for cities, and add only the topical files that matter for that place.
