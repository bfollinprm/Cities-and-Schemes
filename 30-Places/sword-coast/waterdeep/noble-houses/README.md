# Waterdeep — Noble Houses

The eighty-five noble houses of Waterdeep. Tiers (Throne / Court / Body / Dregs) come from the **Great Game** — see [../character.md](../character.md) for tier structure and [../factions.md](../factions.md) for the cross-link table.

Most houses are referenced only in passing; this folder collects the ones the dossier names by family. A house file is a stub-sized handle (≤150 words) for tier, business, and current entanglements.

## Index

- [Adarbrent](adarbrent.md) — Court. Shipping and exploration; rising house with cliff-top ambitions.
- [Agundar](agundar.md) — Court. Mercenary house.
- [Amcathra](amcathra.md) — Throne. Politics, fine wine, sword-smithing.
- [Cassalanter](cassalanter.md) — Throne. Banking; secretly worship Asmodeus.
- [Husteem](husteem.md) — Dregs. Caravans; scorned for scandal.
- [Maerklos](maerklos.md) — Dregs. Mercenary house in disgrace.
- [Tesper](tesper.md) — Throne. Martial spectacle and public duels.
- [Thann](thann.md) — Court. Vintners; Corylus Thann blackmailed by Manshoon.
- [Urmbrusk](urmbrusk.md) — Court. Money-lending; Jelenn Urmbrusk blackmailed by Manshoon.
