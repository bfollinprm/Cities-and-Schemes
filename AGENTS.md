This repository serves as a knowledge store for a D&D campaign.

New information should be stored in well-organized markdown files.
Each folder contains a README.md, which provides the theme of the documents underneath and carries an index and short ~120 character summay of the content of each file.
As you add or delete files, update the index.

Each markdown file should have less than 500 words.
Consider ways to break apart larger files as you write into them, preserving connections as relevant through hyperlinks.
